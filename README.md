# nixos-configuration
My NixOS configuration!

# Why was this repo made?
This repo was made to sync my configuration between devices, but feel free to utilize it to learn something!!

# What does my configuration include?
- Flakes, a technology in Nix.
- Basic education programs, so gnome clocks (for setting an alarm), firefox for online homework sites, and nano (preinstalled by default) for taking notes. Also MPV if you use video lessons.
- Programming tools, so vscodium, and will add the programming languages that I utilize in the future.
- Basic graphical interface, so niri and noctalia shell. My niri configs are NOT INCLUDED for now but may be in the future when I feel like sharing them.
- Social, so Vesktop (a Discord client which comes with Vencord)
- Flameshot and Grim, basic screenshotting tools.
- Terminal eye candy, so something like fastfetch, cava and hollywood.
- System information, so htop.
- Terminal emulator, so ghostty.
- Chromium, included for webapps, as listed in the comment.

# A few small notes
This is supposed to be a desktop configuration, not a server configuration (my server runs NixOS, I will put that out soon).
